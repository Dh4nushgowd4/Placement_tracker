# Placement Tracker Pro

A modern, full-featured placement/job application tracking web app built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Features

- **Dashboard** — Stats overview: Total, Selected, Rejected, Pending, Interviewing, Success Rate
- **Application Management** — Add, Edit, Delete applications with full validation
- **Smart Table** — Search, Filter by status, Sort by any column
- **Analytics** — Circular success rate gauge, Monthly bar chart, Status distribution pie chart
- **Local Storage** — All data persists in the browser
- **Dark Mode** — Premium dark SaaS UI with glassmorphism and glow effects
- **Responsive** — Works beautifully on mobile, tablet, and desktop
